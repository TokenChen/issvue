# issvue

> A client-side github-issue site generator.

## Demo

[cogons.github.io](https://cogons.github.io)

## Clone

``` bash
$ git clone https://github.com/cogons/issvue.git
$ cd issvue
$ npm install
$ npm start

```

## Setup

``` bash
$vi ./src/config.js
```

## Build

``` bash
$ npm run build
$ cd dist

now you can publish the whole dist folder
once and for all.

```
